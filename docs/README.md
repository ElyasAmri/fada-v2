# Documentation

All project documentation organized by topic.

## Structure

- **project/** - Core project specifications and architecture
  - Project overview, specs, architecture, validation checklist

- **experiments/** - Experimental results and summaries
  - **vlm/** - Vision-Language Model testing documentation
  - **vqa/** - Visual Question Answering experiments
  - **integration/** - Integration experiments (FetalCLIP, MedGemma)

- **work_logs/** - Historical work summaries
  - Progress reports and session summaries

- **papers/** - Literature review and research papers
  - Bibliography, literature review, analysis
  - **pdfs/** - Downloaded research papers (gitignored)

- **archive/** - Obsolete or outdated documentation
  - Historical files kept for reference
