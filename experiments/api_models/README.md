# API Models Testing

Tests for cloud-based VLM APIs (Gemini, Grok) on fetal ultrasound images.

## Main Script

- `test_api_vlm_accuracy.py` - Comprehensive API VLM accuracy evaluation

## Usage

```bash
./venv/Scripts/python.exe experiments/api_models/test_api_vlm_accuracy.py
```

## Results

Results are saved to `results/` directory (gitignored).

## Archive

The `archive/` directory contains iterative development versions of test scripts.
These are kept for reference but `test_api_vlm_accuracy.py` is the canonical version.
